# Project Logs

This directory contains append-only monthly project logs.

## Naming

Use monthly files:

```text
YYYY-MM.md
```

## Entry rules

Each non-trivial task should append one concise entry to the current monthly log.

Entries must be factual and should include:

- Date
- Task
- Agent
- Summary
- Files changed
- Verification
- Architectural impact
- Follow-ups

Do not store secrets, API keys, tokens, credentials, or private client data.
